# Lumina-Core

This website is a high-end, immersive digital showcase for a next-generation agricultural technology brand that blends biology, data, and precision engineering. It’s designed not just to inform, but to impress—using cinematic visuals, smooth interactions, and a futuristic interface to reposition agriculture as a cutting-edge industry.

## Tech Stack
- **Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Vanilla CSS
- **Animations:** CSS Keyframes + Intersection Observer
- **Audio:** Web Audio API (Native Oscillator Synth) / High-fidelity MP3

## Features
- **Cinematic Entrance:** Motion-graphic loading sequence.
- **Telemetry HUD:** Real-time scroll-tracking navigation system.
- **Interactive Timeline:** Immersive legacy section with parallax video backgrounds.
- **Horizontal Product Scroll:** Hardware-accelerated smooth-sliding product cards.
- **Custom Soundtrack:** Bespoke ambient audio engine.
